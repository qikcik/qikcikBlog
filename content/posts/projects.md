## Gamejams

### <time>2026.02.01</time>: Mask Off [Kraków gamejam/GGJ 2026]
This narrative (mostly dialogue based) game is metroivania like (especially: Kindergarden) when you stuck in the narrative loop. You dead is  necessary to push narrative forward. created in extended engine from previous game engine with was extended for 3D capability developed in C++ 17/20 using Raylib and rapidXML.\
[gamejam entry](https://globalgamejam.org/games/2026/mask-9-5), [source code](https://github.com/qikcik/GGJ2026)


### <time>2025.02.28</time>: The Last Breath [LubJam/GGJ 2025]
An arcade game where you control an air bubble that constantly floats upwards. The game was built on a custom 2D game engine created entirely during the jam, developed in C++ 17/20 using Raylib and rapidXML, featuring a custom physics and collision engine.\
[gamejam entry](https://globalgamejam.org/games/2025/last-breath-4-0), [game presentation](https://www.youtube.com/watch?v=G_3EmTXrakQ), [editor showcase](https://www.youtube.com/watch?v=-XCZgDah6Bo), [source code](https://github.com/Lap-Luck/LUB2025_LAST_B)


### <time>2024.04.07</time>: Prison Spotlight [Baked GameJam 2024]
A local multiplayer game where one player controls a prisoner trying to escape, while the other tries to prevent the escape using a spotlight. Created in Unreal Engine 5.\
[gamejam entry](https://gamequad.itch.io/prison-spotlights), [game presentation](https://www.youtube.com/watch?v=zmhtSmuZovQ)


### <time>2024.01.28</time>: Sumo Party [KrakJam/GGJ 2024]
A simple local party game with no controller limits—any smartphone can become a gamepad. The game was created in UE5, while the controllers were built using JS+HTML+CSS, utilizing WebSockets for server-controller communication.\
[gamejam entry](https://globalgamejam.org/games/2024/sumo-party-8),  [game presentation](https://globalgamejam.org/games/2024/sumo-party-8)

### <time>2023.01.05</time>: Routing Temple [KrakJam/GGJ 2023]

[gamejam entry](https://v3.globalgamejam.org/2023/games/routing-temple-8),  [game presentation](https://www.youtube.com/watch?v=A_r_tLE626I)

### <time>2022.08.14</time>: Walking Simulator [Arena Gamejam 2022]

### <time>2021.08.15</time>: Hugo wraca do polsatu [MGA Gamejam 2021]
An arcade mine-cart racer inspired by the classic "Hugo" TV show. The player controls the character by dialing specific numbers on a virtual phone dial pad. Created in Godot.\
[game presentation](https://www.youtube.com/watch?v=h2MFukTbRkA)

### <time>2020.11.29</time>: After border of time [HackYeah]
A game created during the COVID pandemic (developed locally in a rented room). The core mechanic revolves around cooperating with the "ghosts" of your own previous attempts to survive in a spacecraft. Created in Unreal Engine 4.\
[game presentation](https://www.youtube.com/watch?v=Rn86SN-1_rA)

### <time>2020.07.26</time>: AlkoTaxi [Slavic GameJam 2020]
A fast-paced, humorous taxi simulator set in the communist era of Poland. The player transports passengers to earn alcohol, which is required to keep the main character going—but drinking too much makes driving almost impossible. Developed  using Godot.\
[game presentation](https://www.youtube.com/watch?v=b2en3QbIxIw), [win32 build](https://drive.google.com/file/d/158VHzJms8rJoyZlTAIyRMWM8Kp3mVHfF/view), [source code](https://drive.google.com/file/d/12RFqgCRtU1QDzn33_krrWJYiWyy7bNX-/view)

### <time>2020.04.06</time>: Janusz Pilnuj Dzieci [ZOSTANWDOMUROBGRY]
A competitive online multiplayer party game created during a 7-day game jam. Players connect via Steam, split into teams, and race to build a sandcastle while sabotaging the opposing team. Developed in a 4-person team (my role: Programmer/Designer) using Unreal Engine 4.\
[game presentation](https://www.youtube.com/watch?v=N2QUwt3tX9M), [win32 build](https://drive.google.com/file/d/17rvmkKoTyyXC_SNShubRy0hZlP_-3Ssg/view), [source code](https://github.com/qikcik/JanuszPilnujDzieci)
